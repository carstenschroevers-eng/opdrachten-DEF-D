# Opdrachten-DEF-D

In deze repository vind je de opdrachten voor DEF-D. 
Design Engineering voor Fysici: Design (DEF-D) wordt gegeven aan eerstejaars studenten technische natuurkunde aan de Technische Universiteit Delft.

We hebben hier de fysieke ontwerpopdrachten en de digitale opdrachten voor DEF-D.

## Fysieke opdrachten

De fysieke opdrachten ga je op de woensdagen maken in groepen die wij voor je indelen.
We gebruiken niet altijd Python notebooks, bij ontwerpopdracht 3 en 6 gebruiken we iets anders!
Voor ontwerpopdracht 3 leveren we hier wel de juiste files alvast.

1. Kalibreer een sensor
2. Ontwerp een sensor
3. Ontwerp een transistor
4. Ontwerp een algoritme
5. Maak een opstelling trillingsvrij
6. Ontwerp een microscoop
